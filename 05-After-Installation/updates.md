# 🔄 System Updates & Maintenance

Keeping your system updated ensures you have the latest security patches and kernel improvements.

Run these commands periodically in your terminal:
    ```bash
    sudo apt update          # Fetches the list of available updates
    sudo apt upgrade -y      # Installs the updates
    sudo apt autoremove -y   # Cleans up old, unused dependencies
