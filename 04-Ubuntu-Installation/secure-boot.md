# 🔐 Secure Boot & Third-Party Drivers

If you re-enabled **Secure Boot** after installation, Ubuntu will prompt you to enroll a MOK (Machine Owner Key).
1. Upon your first reboot, a blue/grey screen (MOK Management) will appear.
2. Select **Enroll MOK** > **Continue** > **Yes**.
3. Enter the password you created during the Ubuntu installation.
4. Select **Reboot**. This allows Ubuntu to load signed drivers (like NVIDIA or Wi-Fi).
