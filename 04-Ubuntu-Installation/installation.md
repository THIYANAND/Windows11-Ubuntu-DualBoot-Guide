# 🐧 Installing Ubuntu 26.04 LTS

1. Insert your bootable USB and restart your PC.
2. Press your **Boot Menu Key** (F12, F11, or F10) during startup to select the USB drive.
3. Select **"Try or Install Ubuntu"**.
4. Once loaded, click **"Install Ubuntu"**.
5. When asked for "Installation Type", select **"Install alongside Windows Boot Manager"** for the easiest experience. 
6. Follow the on-screen prompts for language, keyboard layout, and user account creation.
