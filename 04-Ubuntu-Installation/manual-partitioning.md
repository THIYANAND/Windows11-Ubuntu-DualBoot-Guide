# 🛠️ Manual Partitioning ("Something Else")

If you prefer full control:
1. Choose **"Something Else"** at the installation screen.
2. Select your **Unallocated space**.
3. Click the **+** button.
4. Set the **Mount point** to `/` and use `ext4` journaling system.
5. Ensure the **Device for boot loader installation** points to your main drive (e.g., `/dev/nvme0n1` or `/dev/sda`).
