# 💾 Step 1: Backup Your Data

Before modifying partitions or installing a new operating system, it is **absolutely critical** to back up your important files. While the dual-boot installation process is safe if followed correctly, unexpected power failures or accidental misclicks can result in data loss.

## 1. Personal Files Backup
Copy your important documents, photos, projects, and downloads to a safe location:
*   **External Drive:** USB Flash Drive or External Hard Drive.
*   **Cloud Storage:** Google Drive, OneDrive, Dropbox, or GitHub (for your code).

## 2. Create a Windows System Restore Point
In case you make a mistake in Windows before the installation, a restore point can revert your system settings.
1. Click the **Start Menu** and type `Create a restore point`.
2. Hit **Enter** to open the System Properties window.
3. Select your `C:` drive and click **Configure**. Ensure "Turn on system protection" is selected.
4. Click the **Create...** button, name it something like `Pre-Ubuntu Install`, and wait for the process to finish.
