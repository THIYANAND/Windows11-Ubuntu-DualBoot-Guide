# 🔧 Fixing Boot Issues with Boot-Repair

1. Boot your PC using your **Ubuntu Live USB** (select "Try Ubuntu").
2. Connect to the internet.
3. Open a terminal and install the Boot-Repair tool:
   ```bash
   sudo add-apt-repository ppa:yannubuntu/boot-repair
   sudo apt update
   sudo apt install -y boot-repair
Launch the application:

    ```Bash
        boot-repair

1. Click "Recommended repair" and follow the on-screen prompts.
2. Restart your computer.
